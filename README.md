# 🌦 Weather Dashboard

A modern and responsive weather dashboard built using HTML, CSS, and JavaScript.

The application allows users to search any city and view real-time weather conditions along with detailed weather analytics.

---

## ✨ Features

- 🔍 Search weather by city
- 🌡 Current temperature
- 💧 Humidity
- 🌬 Wind Speed
- 📈 Pressure
- ☀ UV Index
- 🌅 Sunrise & Sunset
- 🌍 Air Quality Index
- 📊 Weather Charts
- 📅 5-Day Weather Forecast
- ⭐ Recent Searches

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
- Open-Meteo Geocoding API
